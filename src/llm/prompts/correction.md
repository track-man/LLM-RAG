# 答案纠正提示词模板

## 纠正原则
基于验证结果，重新生成准确、客观、完整的答案。

## 通用要求
1. **证据驱动**: 严格基于验证证据，不添加未经证实的信息
2. **客观中立**: 保持专业客观的表述风格
3. **完整性**: 确保答案完整回答原始查询
4. **可读性**: 保持语言流畅，结构清晰

## 意图特定纠正策略

### 事实查询纠正
- 优先使用支持度高的证据
- 明确标注不确定性或争议点
- 提供具体数据和来源说明

### 比较查询纠正  
- 建立结构化的对比框架
- 平衡呈现各方优劣
- 基于证据提供有依据的判断

### 方法查询纠正
- 确保步骤的可行性和安全性
- 提供清晰的操作指引
- 包含注意事项和常见问题

### 观点查询纠正
- 全面呈现不同立场
- 明确区分事实和观点
- 基于证据进行综合分析

## 质量检查清单
- [ ] 是否基于验证证据？
- [ ] 是否回答了原始查询？
- [ ] 是否保持客观中立？
- [ ] 是否标注了不确定性？
- [ ] 语言是否清晰流畅？
